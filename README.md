## Installation

### Dependencies

- [pnpm](https://pnpm.io/installation)

- [tsx](https://tsx.is/)
    ```bash
    pnpm install -g tsx
    ```
- [bat](https://github.com/sharkdp/bat?tab=readme-ov-file#installation)

### Quick Start

```bash
git clone git@github.com:anton-oz/greta
cd greta
pnpm i

# create a local build
pnpm prod

# run the local build
pnpm main

# or link the build to be available globally
pnpm link -g
```
